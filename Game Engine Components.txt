Classes
	- Sprites - A class to load and manipulate asset images
	- Sound - A class to load and manipulate SFX
	- Music - A class to load and manipulate all music files
	- Font - A class to load and manipulate texts and fonts
	- Animated Sprites - A class to animate assets 
	- Game Object - A giant class that extends all the above classes and more
	 	 - Contains properties of game objects that are going to be added on to the game
		 - Allows to create object w or wout images/sounds/music 
		 - Keeps track of the game object it self (xy Pos, xyRot, xyTrans/xyScale)

Advanced Classes
	- Collision Detector - A class to manipulate and detect collisions
	- Particle System - A class to manipulate particles 
	- Camera - A class to manipulate viewport
	- UI - A class to load and manipulate UI components
	!- Light - A class to eluminate and manipulate light
	- Environment - A class to load and manipulate surrounding environment
	- Rigid Body - A class to simulate and manipulate physics of game objects 
	# Physics Engine - A class to apply physics onto game objects
